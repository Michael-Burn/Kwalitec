# Day 1: 

## Study Context

- Date: 13 July 2026
- Planned study duration: 180 minutes
- Actual study duration: Session 1 - 45 minutes
- Topic studied: Generalised linear Models

## Recommendation

- What Kwalitec recommended: Topic 4.2 Generalised linear Models
- Did it feel appropriate? (Yes / Partly / No): Yes
- Why? This is my correct position in the course interms of prior Study progress

## Study Flow

- Was starting the study session smooth? Yes technicaly but I didn't manage to concentrate because I only got approx 3.5 hours of sleep last night
- Any friction? Friction was mainly due to the UI not being polished enough. In the Curriculum Raod Map section, there are some topics that are not assigned weights. There are only 5/20 topics assigned with weights

## Educational Value

- Did the recommendation improve today's study?
- Did the explanation help? The descriptions of the missions needs improving for example  - First milestone was "Study and practice Understand and use generslised linear models for 62 minutes" - I got what was meant but it was not written properly.

- Did anything feel misleading or uncertain? The topic is long and most likely take more than one study session. The missions therefore should be more intricate rather than 4.2 maybe say part of 4.2 or 4.2.1-4.2.2 etc as an example.

## UI / Experience

- One thing that felt premium: The mission was straight to the point and linked to the syllabus which I appreciate. The plan was also decisive and straight to the point

- One thing that felt unfinished: The text in the daily missions is not clearly legible (too faint). When I hover on the "Continue Studying" button on the main dashboard, there is some text that is appearing which is distorting the view. The "Today's recommendation" section has an error "Why: warrant_honesty, diagnostic_clarity, no_mid_preparedness_theatre feasibility/ample cold_start; thin_warrant; not_yet_knowable; decision_warrant:cold_start; confidence:cold_start"
- app is jumping straight to the main dashboard skipping login page when I click its link
- In missions, the mark as complete button is not working, The progress bar is not working.

## Reflection

- One key observation from today:

--- I needed enough sleep to be able to focus today
--- Kwalitec should inform the student of the best setting in terms of environment to focus
--- Require a way to add another user so that my friend can also participate in the test runs
