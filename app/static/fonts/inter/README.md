# Inter (self-hosted)

Latin subset WOFF2 files for product UI (UX-001).

| Weight | File |
|--------|------|
| 400 | `inter-latin-400-normal.woff2` |
| 500 | `inter-latin-500-normal.woff2` |
| 600 | `inter-latin-600-normal.woff2` |
| 700 | `inter-latin-700-normal.woff2` |

Source: [Fontsource Inter](https://github.com/fontsource/font-files) (Google Inter, SIL Open Font License 1.1).

Loaded via `app/static/css/fonts.css` — do not load Inter from a CDN in product shells.
